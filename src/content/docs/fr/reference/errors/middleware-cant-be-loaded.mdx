> **MiddlewareCantBeLoaded**: Le middleware a généré une erreur pendant qu'Astro essayait de le charger.

## Qu'est-ce qui ne va pas ?
Lancé en mode développement lorsque le middleware génère une erreur lors de la tentative de chargement.

Par exemple :
```ts
import {defineMiddleware} from "astro:middleware";
throw new Error("Error thrown while loading the middleware.")
export const onRequest = defineMiddleware(() => {
  return "string"
});
```
